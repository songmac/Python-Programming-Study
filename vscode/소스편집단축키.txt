Alt + Up : 행을 위로 이동

Ctrl + Alt + Up : 커서를 위에 추가

Ctrl+ ] : 라인 들여쓰기
Ctrl+ [ : 라인 내어쓰기

Ctrl + Up : 스코롤 업
Alt + PageDown : 아래로 페이지 스크롤
Alt + PageUp : 위로 페이지 스크롤
Ctrl + Shift + [ : 영역 접기
Ctrl + Shift + ] : 영역 펼치기
